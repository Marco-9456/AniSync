Release 1.5.0

This release strengthens AniSync's social and connected navigation experience.

After the profile redesign in 1.4.0, this update expands interaction depth: activities now open in a dedicated details flow, profile sections gained better pagination and direct navigation hooks, deep links cover more AniList URL surfaces, and all widgets were redesigned for a more cohesive modern presentation.

Added

Activity & Profile

  • Activity Detail Screen - Added a dedicated activity detail route with richer activity content rendering.
  • Profile Messaging - Added a message composer on public profiles backed by AniList SaveMessageActivity.
  • Profile Tab Pagination - Social and Reviews tabs now paginate for smoother browsing on large profiles.
  • Connected Profile Navigation - Profile social/review entries now open related threads, comments, and review detail pages directly.
  • Profile Pull-to-Refresh - Added pull-to-refresh support across profile flows.

Navigation & Notifications

  • Expanded Deep Links - Added routing for user profiles, reviews, and forum comments from AniList links.
  • Social Activity Alerts - Added social activity notification channel, preferences, and dispatch integration.

Widgets

  • Material 3 Widget Refresh - Redesigned all Glance widgets with modern card-based layouts.
  • Weekly Day Picker - Added day selection and improved schedule fetching for Weekly Calendar.
  • Up Next Streaming Links - Added streaming link support in airing data for the Up Next widget.
  • Updated Widget Previews - Refreshed launcher preview layouts to match the new widget visuals.

Changed

  • Transition Coverage - Extended shared elements across detail/grid flows and stabilized return behavior in Library/Discover.
  • Card Design Refresh - Updated activity/forum/content cards for denser, cleaner readability.
  • Build Stack Upgrade - Migrated to AGP 9.1.0, SDK 37 targets, and AGP 9-compatible APK naming flow.
  • Database Schema - Advanced Room to version 12 with auto-migration (11 -> 12).

Fixed

  • Rich Text Edge Cases - Improved mixed markdown/HTML link handling and inline grouping behavior.
  • Transition Stability - Reduced shared-transition return glitches across key navigation flows.
